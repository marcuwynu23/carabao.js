Why Carabao.js?

Built for developers who want structure without the overhead. A familiar Express foundation with TypeScript superpowers.

CLI Scaffolding

Generate a complete MVC project with a single command. Routes, controllers, middlewares — all wired and ready.

TypeScript-First

Full type safety from request to response. Catch errors at compile time, not runtime.

Express-Based

Familiar API on a battle-tested foundation. No new paradigm to learn — just Express, made cleaner.

Auto-Wired Routing

Define your routes in one place. Controllers are wired automatically — no manual setup.

Flexible Module System

Supports both CommonJS and ESM. Use whatever your project needs.

Zero Config

Sensible defaults out of the box. Easy to override when you need more control.
