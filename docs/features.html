Features

Everything you need to build production-ready Node.js APIs — nothing you don't.

Core Features

CLI Scaffolding

Generate a complete MVC project with a single command. No manual folder setup, no boilerplate copying.

TypeScript-First

Full type safety from request to response. Strict mode enabled by default.

Express Foundation

Built on Express 4.x. Use any Express middleware, template engine, or ecosystem library.

Auto-Wired Routing

Define routes declaratively. Controllers are wired automatically — no manual app.get() chains.

Dual Module Support

Ship both CommonJS and ESM builds. Use whichever module system your project requires.

Zero Configuration

Sensible defaults for Express, TypeScript, and project structure. Override when you need to.

Environment Ready

dotenv integration built-in. .env file generated with every scaffold.

Jest Integrated

Test setup included with every scaffold. Write and run tests immediately.

Supported Frameworks & Tools

Carabao.js scaffolds projects that work with the most popular Node.js ecosystem tools:

Express TypeScript Jest dotenv ts-jest supertest ESBuild Node.js 18+ npm yarn pnpm

Export Formats & Integration

Carabao.js itself is available as:

  • CommonJS — require('carabao')
  • ESM / TypeScript — import { createApp } from 'carabao'
  • CLI — npx carabao create my-app

The framework exports both Application (class-based) and createApp (functional) APIs, giving you flexibility in how you structure your server code.
